Ruby on Bare Metal

CRuby 4.0.2 running on a bare-metal x86_64 kernel in your browser via QEMU WebAssembly

Boot a bare-metal kernel running CRuby directly in your browser.

Requires a modern browser with SharedArrayBuffer support.

QEMU WebAssembly port powered by ktock/qemu-wasm (GPLv2). Source code: ruby-on-bare-metal (GPLv2).
